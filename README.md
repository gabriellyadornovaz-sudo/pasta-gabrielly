# 🌍 Clima Urbano vs. Clima Rural (Y2K Weather Station) 💿👾

Este repositório contém uma aplicação web interativa baseada no impacto do uso do solo nos microclimas locais (Urbano vs. Rural). O projeto conta com uma interface inspirada na estética futurista dos anos 2000 (Y2K), trazendo dados em tempo real e efeitos sazonais dinâmicos.

---

## 📋 Descrição do Projeto

O clima de uma região é diretamente afetado pela urbanização. Este projeto estuda o fenômeno das **Ilhas de Calor Urbanas (ICU)** em comparação com o equilíbrio das zonas rurais, exibindo as diferenças climáticas através de uma aplicação interativa que consome dados mundiais ao vivo.

### Características Monitoradas:
* **Área Urbana:** Ilhas de calor, menor umidade devido à pavimentação e poluição concentrada.
* **Área Rural:** Temperaturas mais amenas, maior taxa de evapotranspiração e ar limpo.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** & **CSS3** (Layout responsivo com grade cibernética)
* **JavaScript (ES6+)** (Manipulação assíncrona do DOM e animações de partículas)
* **OpenWeatherMap API** (Dados meteorológicos e previsões em tempo real)
